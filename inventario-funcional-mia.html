Resumen Ejecutivo

MIA es la asistente inmobiliaria con IA de Ciencuadras (Servicios Bolivar S.A., NIT 900.311.092-7). Plataforma digital para arriendo, venta y alquiler de inmuebles en Colombia.

URL: https://www.ciencuadras.com/mia-asistente-inmobiliaria
Orquestacion: Langfuse (prompt management + observabilidad)
Modelo: Google Gemini

Hallazgos Clave

  • LangGraph Swarm: MIA usa patron Swarm donde agentes se transfieren control entre si
  • Supervisor Agent: Orquesta el flujo completo (equivalente al Router en Langfuse)
  • MCP Protocol: ciencuadras-mcp-inmuebles-ia-ms implementa Model Context Protocol para tools de busqueda
  • MongoDB Checkpoints: Estado de conversaciones persistido (multi-turno con estado)
  • RedisVL: Redis como vector store para RAG de FAQs (no Pinecone ni PgVector)
  • Libreria transversal: ia-transversal-langchain-python-lib compartida a nivel organizacional
  • Doble observabilidad: Datadog (APM/infra) + Langfuse (trazas LLM)
  • Flask (no FastAPI): Desviacion del stack aprobado organizacional
  • Java legacy: ciencuadras-modelos-ia-ms es Spring Boot/Gradle (v1 del sistema, aun activo)
  • Cada agente tiene su propio README.md con documentacion interna

Reglas de Negocio

  • 1. Arriendo vs Compra: Si presupuesto < $100M o rango mensual → Arriendo. Si dice nuevo/estrenar + intencion compra → Compra.
  • 2. VIS: proyecto=true, transaccion=venta, precio maximo $213.525.000
  • 3. Cuota inicial: Leasing 20%, Hipotecario 30%
  • 4. Horarios visita: Lun-Vie 8am-5pm, Sab 8am-12pm
  • 5. Datos obligatorios lead: nombre, identificacion, tipo doc, telefono, email
  • 6. Politica privacidad: Siempre mencionar antes de capturar datos
  • 7. Contactos inmobiliaria: Solo DESPUES de save_lead_to_zoho exitoso
  • 8. Formato WhatsApp: Un asterisco para negritas, URLs texto plano
  • 9. Sitios de interes: Salud, Educacion, Estacion Servicio, Parque, Transporte, Centro Comercial, Supermercado
  • 10. Validacion financiera: Comparar preaprobado vs valor inmueble
  • 11. Urgencia: Si plazo > 3 meses, redirigir a proyectos vivienda nueva
  • 12. Co-deudor: Si credito insuficiente, sugerir co-deudor o mayor cuota inicial
